## Board Dimensions

![Board Dimensions](generated/pcb/placeholder_dimensions.svg)
*Technical drawing with exact measurements*

### Key Dimensions

| Dimension | Value |
|-----------|-------|
| Width | 40.64mm (8HP) |
| Height | 128.5mm (3U) |
| PCB Thickness | 1.6mm |
| Mounting Holes | M3 (3mm) |
| Component Height (max) | 35mm |

### Mounting Hole Positions

Holes are positioned according to Doepfer eurorack standard:
- **Vertical spacing:** 105mm (center to center)
- **Horizontal position:** 3mm from left/right edges
- **Hole diameter:** 3.2mm (for M3 screws)
