# Sourcing Parts

Where to buy components for this module.

## Complete Kits

Currently no commercial kits available. See [BOM](bom.md) for self-sourcing.

## PCB Only

Order PCBs from:
- [JLCPCB](https://jlcpcb.com) - Low cost, fast shipping
- [PCBWay](https://www.pcbway.com) - Alternative manufacturer
- [OSH Park](https://oshpark.com) - USA-based, purple boards

## Component Suppliers

See [BOM page](bom.md#recommended-suppliers) for detailed supplier information.

## Front Panels

If using a custom front panel:
- DIY: Order from Ponoko, SendCutSend
- Commercial: Check with panel manufacturers

## Bulk Ordering

For group buys or multiple modules, consider:
- Shared PCB orders (MOQ 5 pcs at JLCPCB)
- Bulk component pricing (Digikey, Mouser)
- Assembly service for larger quantities
